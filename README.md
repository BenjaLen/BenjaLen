## 🧠 sobre mí

Soy un desarrollador enfocado principalmente en Python y herramientas interactivas.  
Me gusta crear proyectos que mezclen utilidad con personalidad, desde bots para Discord hasta aplicaciones completas que todavía no tienen utilidad definida.

La mayoría de mis proyectos nacen de:
- "¿Por qué esto no existe?"

- o simplemente una idea random a las 3 AM.

- APIs
- back-end
- diseño de aplicaciones
- automatización de procesos
